# 46hunnybun Project

This project contains a Jupyter notebook for medical image classification using ResNet50/DenseNet121.

## Setup

### Prerequisites

-   **Python 3.10** (The project is tested with this version)

### Requirements

To install the dependencies locally:

```bash
pip install -r requirements.txt
```

## Data

The data is expected to be in the `4600 Data/` directory. Ensure you have the data available locally.

## Usage

To run the Jupyter notebook, execute:

```bash
jupyter notebook
```
